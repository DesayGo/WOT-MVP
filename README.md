# WOT-MVP：战斗助手 / 复盘教练（不接管操作）

这是一个面向《坦克世界（World of Tanks, WOT）》的 **MVP 原型**，目标是做“决策辅助 + 训练提升”，而不是直接替玩家执行操作。

## 当前实现（前端原型）

本仓库提供了一个可直接打开的前端页面（`app/index.html`）：

- 录像文件输入：支持拖拽/选择 `.wotreplay` 文件
- 战场感知面板：
  - 敌我分布摘要
  - 火力密度等级
  - 敌方可见坦克威胁排序
- 实时建议：
  - 当前姿态建议（卖头/侧摆/后撤）
  - 穿深风险提示
  - 转场点建议
- 对线建议：
  - 敌车弱点速查
  - 装填时间窗口提醒（估算）
- 战后复盘：
  - 死亡原因分类
  - 时间轴回放（关键节点）
  - 更优决策分支建议
- 地图标注：
  - 优先加载地图俯视底图资源（`app/maps/*.svg`），并叠加战术标注
  - 标注“危险点 / 推荐转场点”
- 录像地图自动识别：
  - 优先从 `.wotreplay` 文件名识别地图
  - 回退到录像内容片段（UTF-8/UTF-16）识别，失败时允许手动选择
- 时间轴建议：
  - 支持拖动时间轴（0~8 分钟）
  - 根据开局/中期/残局给出分时策略

> 说明：当前版本已支持“地图名自动识别”的前端启发式能力，但仍不是完整的 `.wotreplay` 协议级解析。

## 如何运行

### 方式一：直接打开

直接用浏览器打开 `app/index.html`。

### 方式二：本地静态服务（推荐）

```bash
cd /workspace/WOT-MVP
python3 -m http.server 8000
```

然后访问：<http://localhost:8000/app/index.html>

## MVP 后续演进建议

1. **Replay 解析层**
   - 接入 `.wotreplay` 解析器（例如 Python 服务）抽取：事件流、车辆坐标、击穿记录、伤害日志。
2. **策略引擎层**
   - 规则 + 轻量模型混合：按地图、坦克类型、对位关系给出建议。
3. **可视化层**
   - 叠加真实地图坐标，支持时间轴拖动与热区回放。
4. **训练闭环**
   - 复盘报告结构化输出（站位、换血、控线、转场时机）。
